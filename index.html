Guia Rapida de Comandos Git

git init

Inicializa un nuevo repositorio Git local en el directiorio actual

git status

Muestra el estado actual del repositorio, archivos modificados y no rastreados

git add .

Agrega todos los archivos al área de preparación (staging)

git commit -m "mensaje"

Guarda los cambios agregados con un mensaje descriptivo.

git push

Envía los commit locales al repositorio remoto.

git pull

Obtiene y fusiona los cambios del respositorio remoto
